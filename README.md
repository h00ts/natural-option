# Natural Option

Wordpress theme for Natural Option.

Built by TFC Marketing.

## Requirements

### Plugins
* Woocommerce
* Elementor Pro

## Installation

1. Download this repository as a .zip
2. Upload it via Wordpress: Appearance > Themes > Add New > Upload Theme
