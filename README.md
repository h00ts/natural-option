# Natural Option

Wordpress theme for Natural Option.

Built by TFC Marketing.

## Requirements

### Plugins
* Woocommerce
* Elementor Pro

## Installation

1. Download this repository and unzip
2. cd into directory and npm install
3. Run gulp copy-assets
4. Copy folder to wp-content/themes
